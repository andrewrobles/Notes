### vim

motions
- `h`, `j`, `k`, `l` left, down, up, right 
- `b`, `w` left word, right word

editing
- `d`, `y` cut, copy
- `dd`, `yy` cut line, copy line
- `u`, `^r` undo, redo
- `p` paste

modes
- `i`, `a` insert left, insert right
- `v`, `^v` visual, visual line
